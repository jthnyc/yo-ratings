# YearOne Movie Ratings

YO-Ratings is a single page application to rate movies\
[Deployed Site Link](https://yo-ratings-git-main.joannathuang.vercel.app/)

## Getting Started

1. Fork and clone this repo
2. npm install
3. npm start
4. Visit http://localhost:3000
